Games and projects

Gravity

A semi accurate simulation of a three-body gravitational system. I made this as my high-school "graduation project", hence the focus on math/physics.

SchillerSegregation

A simple simulation of cells following Schiller's ruleset for segregation. Parameters must be edited in code to display different scenarios. This was an assignment I made for the first programming course at Chalmers.

Rolf-pong

A pong game I made in triute to Prof. Rolf at Chalmers.

SMHIJson
